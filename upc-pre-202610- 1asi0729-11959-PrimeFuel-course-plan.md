Este curso introduce a estudiantes de 12 a 17 años a los fundamentos de la programación con Java y la Programación Orientada a Objetos (POO). ¡No requiere descargas! Solo abre tu navegador y empieza a programar.

# 🎬 Secuencia de Lecciones

## Lección 1: ¿Qué es Java y la Programación? (~5 minutos)

- **Descripción:** Descubre qué es programar y por qué Java es uno de los lenguajes más utilizados en el mundo.
- **Enlace:** Ver la lección
- **Conclusiones clave:** Los programas son instrucciones para computadoras; Java se utiliza en aplicaciones, videojuegos y sistemas empresariales.
- **Empieza a programar:** Abrir Replit - Hola Mundo

---

## Lección 2: Variables y Tipos de Datos (~8 minutos)

- **Descripción:** Aprende a almacenar información utilizando variables y tipos de datos básicos.
- **Enlace:** Ver la lección
- **Conclusiones clave:** Uso de `int`, `double`, `boolean` y `String`.
- **Práctica:** Clic para programar 👈 ¡Sin registro!

---

## Lección 3: Operadores y Estructuras de Control (~10 minutos)

- **Descripción:** Realiza cálculos, toma decisiones y automatiza tareas con condicionales y bucles.
- **Enlace:** Ver la lección
- **Conclusiones clave:** Uso de operadores, `if`, `else`, `for` y `while`.
- **Práctica:** Clic para programar 👈 ¡Inicio inmediato!

---

## Lección 4: Métodos y Entrada de Datos (~10 minutos)

- **Descripción:** Organiza mejor tu código creando métodos y obteniendo información del usuario.
- **Enlace:** Ver la lección
- **Conclusiones clave:** Métodos reutilizables y lectura de datos mediante `Scanner`.
- **Práctica:** Clic para programar 👈 ¡Sin registro!

---

## Lección 5: Introducción a la Programación Orientada a Objetos (~10 minutos)

- **Descripción:** Aprende qué son las clases y los objetos mediante ejemplos sencillos.
- **Enlace:** Ver la lección
- **Conclusiones clave:** Diferencia entre clases y objetos; creación de atributos y métodos.
- **Práctica:** Clic para programar 👈 ¡Empieza a crear objetos!

---

## Lección 6: Constructores y Encapsulación (~8 minutos)

- **Descripción:** Mejora tus clases utilizando constructores, getters y setters.
- **Enlace:** Ver la lección
- **Conclusiones clave:** Encapsulación y protección de datos dentro de los objetos.
- **Práctica:** Clic para programar 👈 ¡No necesitas instalar nada!

---

## Lección 7: Arrays y Proyecto Final (~12 minutos)

- **Descripción:** Utiliza arreglos junto con objetos para desarrollar una aplicación sencilla de registro de estudiantes.
- **Enlace:** Ver la lección
- **Conclusiones clave:** Manejo básico de arrays, objetos y lógica de programación.
- **Proyecto final:** Crea tu aplicación Java 👈 ¡Guarda y comparte!

---

## Lección 8: Buenas Prácticas y Próximos Pasos (~7 minutos)

- **Descripción:** Aprende errores comunes de principiantes y cómo seguir creciendo como programador.
- **Enlace:** Ver la lección
- **Conclusiones clave:** Leer mensajes de error, escribir código ordenado y practicar constantemente.
- **Comparte tu trabajo:** Envía tu creación
