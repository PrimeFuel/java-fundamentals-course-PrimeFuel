## 🤔 ¿Qué es programar?

"Antes de hablar de Java, primero debemos entender qué significa programar.

Programar consiste en darle instrucciones a una computadora para que realice una tarea.

Piensa en una receta de cocina.

Si quieres preparar una pizza, debes seguir una serie de pasos:

1. Preparar la masa.
2. Agregar los ingredientes.
3. Hornear.
4. Servir.

La computadora funciona de manera parecida.

Necesita que le indiquemos exactamente qué hacer y en qué orden hacerlo.

A esa lista de instrucciones la llamamos programa."

---

## 💻 ¿Qué es Java?

"Java es un lenguaje de programación.

Un lenguaje de programación es una forma de comunicarnos con una computadora.

Así como las personas utilizan diferentes idiomas para comunicarse, los programadores utilizan diferentes lenguajes para crear programas.

Java es uno de los más populares del mundo porque es relativamente sencillo de aprender y se utiliza en muchísimos lugares."

---

## 🎮 ¿Dónde se utiliza Java?

"Probablemente ya has usado algo creado con Java sin darte cuenta.

Java se ha utilizado para crear:

* Aplicaciones de escritorio.
* Sistemas empresariales.
* Aplicaciones Android.
* Servidores y servicios web.
* Herramientas educativas.
* Algunos videojuegos famosos como Minecraft."

"Millones de personas utilizan software desarrollado con Java todos los días."

## 🧠 ¿Por qué aprender Java?

"Java es un excelente lenguaje para comenzar porque enseña conceptos que podrás utilizar más adelante en muchos otros lenguajes.

Al aprender Java desarrollarás habilidades como:

* Resolver problemas.
* Pensar de manera lógica.
* Dividir tareas complejas en pasos sencillos.
* Crear tus propios programas."

"Estas habilidades son útiles incluso fuera de la programación."

---

## 🚀 ¿Qué construiremos en este curso?

"Durante este curso aprenderemos a:

* Guardar información usando variables.
* Tomar decisiones con condiciones.
* Repetir acciones automáticamente.
* Crear nuestras propias funciones.
* Construir objetos y clases utilizando Programación Orientada a Objetos.
* Desarrollar una pequeña aplicación final."

"Y todo esto sin instalar programas complicados."

## 🎯 Primer contacto con Java

> **Nota visual:** Abrir JDoodle.

"Este será nuestro entorno de trabajo.

Aquí podremos escribir código Java directamente desde el navegador.

Más adelante aprenderemos exactamente qué significa cada línea, pero por ahora veamos algo muy sencillo."

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hola Mundo");
    }
}
```

"Este programa simplemente muestra el mensaje 'Hola Mundo' en pantalla.

Es la tradición con la que millones de programadores han comenzado su camino."
